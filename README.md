# Towems Cosmetics and Spa Website

## Overview

**Towems Cosmetics and Spa** is a simple website built using **HTML, CSS, and JavaScript** to showcase hair and beauty products offered by the business.
The website provides information about available products and services and helps customers learn more about the brand.

## Features

* Product showcase for hair and beauty products
* Simple and responsive design
* Clean user interface
* Built with basic web technologies

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
towems-cosmetics-spa/
│
├── index.html
├── style.css
├── script.js
├── images/
└── README.md
```

## Installation / Usage

1. Clone the repository

```
git clone https://github.com/your-username/towems-cosmetics-spa.git
```

2. Open the project folder.

3. Run the website by opening **index.html** in your browser.

## Purpose

This project was created to provide an online presence for **Towems Cosmetics and Spa**, allowing customers to explore available beauty and hair products.
